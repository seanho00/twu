Discussion

  1. Look for candidate objects, identify instance variables and methods, refine the methods while keeping an eye out for new classes or methods.
  2. Encapsulation
    The variables and implementation of methods are contained in the class definition. Client code only needs to know what the methods do, not how they are implemented.
    Polymorphism
    The actual code executed by a method call is determined by the class of the object that the method is being called on. Thus a single line of code may invoke different methods from one call to the next: 
    for obj in objects:
        obj.someMethod()
    Each iteration of this loop may invoke a different form of someMethod
    Inheritance
    Behavior defined in a superclass can be used by a subclass.
