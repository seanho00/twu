The problems in Chapter 12 are open-ended and can serve as sources of significant end-of-term exercises. Three completed examples are provided here.

bridge contains a non-graphical card game type program. It deals four Bridge hands, counts the points in each hand and determines whether each hand can open.

blackjack is an interactive game to play Blackjack. There is a simple text-based prototype and a full graphical version.

hangman implements the spelling game Hangman. This directory includes both a text-based and graphical version. This particular program is an actual example of a project completed by a CS1 student.
