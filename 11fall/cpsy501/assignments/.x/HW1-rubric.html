  2. IQ and English Grade (5 marks)

    One question this research addresses is whether intelligence scores are strongly associated with "English course grade" for the population of 9th grade students from which this sample was drawn. What kind of correlation coefficient provides the correct strategy for answering that question? Calculate the proper result using SPSS and indicate whether the result can be accurately described as "large" using the convention described in the textbook.
    [2pts]: non-parametric correlation coefficient, either Spearman's rho or Kendall's tau. If they choose Kendall's tau and don't cite the textbook or give some justification for it over Spearman, ask them why they chose it.
    [1pt]: effect size using Spearman: large. Using Kendall's tau: medium.
    How would your analysis strategy have changed if you were correlating "English course level in 9th grade" with IQ?
    [1pt]: no change; same non-parametric strategy applies.
    [1pt]: effect size with English course level, using Spearman: medium. Using Kendall's tau: small.

    Extra positive comments for indications that student read SPSS help files and/or textbook for further details on Spearman's rho or Kendall's tau: e.g., addressing tied ranks, etc.
