Description

 This is a course in the elements of programming with one or more object oriented programming languages. Students will be introduced to the general concepts of object oriented programming and will learn one object oriented language in sufficient detail to be able to produce substantial programs.

Objective

 The objective of the course is to make students familiar with the main concepts of OOP. The programming language JAVA which incorporates the principles of OOP with the practice of networking and the Internet will be used almost exclusively. After taking this course students should be able to write substantial programs in JAVA and other OOP languages.

Prerequisites

 CMPT 140 (145 preferred) or an equivalent course or working experience using a procedural language such as Pascal, Modula-2, Oberon, Python, C, or C++.

Required Text

 "Java How to Program", by Deitel and Deitel, 6th ed., Prentice Hall, New Jersey, 2005. ISBN 0-13-128933-0.

Topics

 We will be using Sun's Eclipse development environment, using the J2SE 5.0 platform (Java 1.5.0). This is installed on the computer lab PCs. You may also use your own computer, as long as you have Java at least version 1.4.
