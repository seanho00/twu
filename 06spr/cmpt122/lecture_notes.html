Week 2

  • Unit K:

    • Action Queries:

      • Make-Table Query:
        • view a Query in Design View
        • select Query Type list arrow in toolbar, select Make-Table Query
        • enter a name for the new table
        • Click Run button in toolbar
      • Append Query
      • Delete Query:
        • query design: only show table we want to delete from
        • Add a column to query design that has "TableName.*" in "Field:"
        • Add a column to query design indicating criteria to select records to delete
        • Select Delete Query from Query Type toolbar button
        • Observe new "Delete:" row of query design; "From" and "Where"
        • Click Run button, observe fewer records left
      • Update Query:
        • Query Type: Update Query
        • Enter "[Cost]*1.05" in "Update To:" for Cost

    • Join Properties

      • Relationships: make/recall one-to-many link
      • view a Query in Design View, select two linked tables, observe link
      • Right-click the join line, Properties

    • Find Unmatched

      • New Query dialog, select Find Unmatched Query Wizard
      • Choose two tables with a relationship: First table is where results are taken from, second table is used to select
      • Choose fields for display
      • go to query Design View to see what the wizard did

  • Unit L: Advanced Forms

    • Toggle button:
      • view form in Design View
      • right-click a check-box widget
      • "Change To" / "Toggle Button"
      • Mention: Text box / List box / Combo box
      • Mention: Check box / Toggle button / Option button
    • Form Widget Properties:
      • Select a widget
      • View / Properties / Other
      • fill in help text in "ControlTip Text" property
      • Also try "Status Bar Text" property
    • Conditional formatting:
      • view form in Design View
      • select widget(s), Format / "Conditional Formatting"
      • try a "Field has Focus" Condition
      • go to Form View and play with focus
    • Tabs
      • view form in Design View
      • Toolbox toolbar / Tab Control button
      • two tabs by default; click tab to select
      • Add widgets to each tab, view in Form View
    • Charts in Reports
      • new Report, Chart Wizard
      • double-click chart to edit
      • double-click chart bbox for widget properties
